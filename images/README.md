# Cartella Images

Questa cartella contiene tutte le immagini utilizzate nel sito web.

## Sottocartelle consigliate:
- `hero/` - Immagini per le sezioni principali
- `products/` - Immagini dei prodotti/servizi
- `team/` - Foto del team
- `blog/` - Immagini per articoli del blog
- `icons/` - Icone e loghi

## Formati supportati:
- JPG/JPEG - Per fotografie
- PNG - Per immagini con trasparenza
- SVG - Per icone e grafici vettoriali
- WEBP - Per ottimizzazione web

## Ottimizzazione:
- Comprimi le immagini prima dell'upload
- Usa formati appropriati per ogni tipo di contenuto
- Mantieni le dimensioni ragionevoli per caricamento veloce
